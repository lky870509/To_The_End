"To the End"

플랫폼:PC
장르:RPG
타겟 플레이어:RPG를 좋아하는 전연령

To the End는 불이 사라진 세상 속에서 여신의 성물을 모아 여신으로 부터 불을 되찾는다는 스토리의 RPG 게임입니다.

캐릭터는 남 녀 둘로 나뉘어 있으며,
